# sems-yeti

sems-yeti is a part of project [Yeti]

## Build (Debian 8/9)

### install build dependencies

```sh
# aptitude install libsems1-dev libyeticc-dev
```

### get sources & build

```sh
$ git clone git@github.com:yeti-switch/sems-yeti.git
$ cd sems-yeti
$ ./package.sh
```

[Yeti]:http://yeti-switch.org/
